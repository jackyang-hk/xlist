## changelog

### V3.0.0

- 原始数据格式调整，增加数据样式等描述
- 重用dom数固定，通过infiniteElements配置
- 新增DataSet数据集
- renderHook参数调整，无需return
- 支持任意高度元素回收
- 支持position:sticky
- 性能优化
- panstart事件定义为发生移动才出发，区分touchstart
- 支持同步渲染
- 移除autoRender配置，默认不渲染，需要手动render()
